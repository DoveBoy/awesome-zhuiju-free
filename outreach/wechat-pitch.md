## 文章草稿（可直接提供给对方）

**标题：**
```
每天自动体检的追剧资源导航，再也不怕资源失效了
```

**正文：**

```
追剧的朋友应该都遇到过这个问题：

收藏夹里存了一堆影视站，每次想看剧打开，不是挂了就是满屏广告弹窗。好不容易找到一个能用的，下次又挂了。

直到发现了这个开源项目：Awesome Zhuiju Free。

**它解决的核心问题，就是让你随时知道哪些资源今天可用。**

每天早上，项目会自动检测收录的所有资源是否能访问，用颜色直接标注结果：

- 🟢 可访问：正常可用
- 🟡 访问受限：存在但有限制
- 🔴 无法访问：今天挂了

不用自己一个个去试，打开页面就知道今天该用哪个。

所有收录资源均经过人工筛选，只有免费且无广告的内容才会入库，有弹窗广告、需要付费的站点不会被收录。

---

**收录了什么？**

目前 60+ 个资源，覆盖：

- 在线影视（直接浏览器看）
- 影视APP（手机 / 电视）
- 网盘资源搜索
- TVBox / 影视仓配置地址（电视盒子必备）
- 磁力 & BT
- 字幕资源

每个资源都有评分和风险标注，版权风险如实标注，不掩盖。

---

**怎么用？**

不需要注册，不需要安装：

- 直接访问：https://zhuiju.me
- GitHub 项目：github.com/laoma2053/awesome-zhuiju-free
- 国内 Gitee 镜像：gitee.com/laoma2053/awesome-zhuiju-free

完全免费，无广告，开源，社区维护。

发现好用资源也可以在 GitHub 提 Issue 推荐，有维护者审核发布。
```
